# BOOTH-Lens API

This is the backend for BOOTH-Lens, deployed on Hugging Face Spaces.
It provides image search functionality using CLIP and Qdrant Cloud.

## Deployment Details
- **SDK**: Docker
- **Backend**: FastAPI
- **Search**: CLIP + Qdrant Cloud
- **Storage**: Cloudflare R2
